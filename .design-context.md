# TheraNotes AI — Design Context

## Product
Clinical report generation tool for Australian occupational therapists writing NDIS Functional Capacity Assessment (FCA) reports. Clinicians paste notes, AI generates a multi-section clinical report, then they edit and refine it in a document workspace.

## Users
Allied health professionals — primarily occupational therapists. Non-technical. Use iPads and desktops in clinical settings. Trust and professionalism are paramount. They are submitting these documents to government funding bodies (NDIS).

## Aesthetic
Warm, quiet, minimal. Clinical without being sterile. The design draws from Banko-inspired fintech aesthetics — off-white backgrounds, near-black ink, restrained use of color. Blue accent reserved exclusively for AI/interactive moments. Serif fonts for the clinical document body (Georgia), sans-serif (Geist) for the UI chrome.

## Brand Colors (established)
- Background: `#F6F5F2` (warm off-white)
- Ink: `#1C1B22` (near-black)
- Muted: `#6b6a74` / `#9a9a9f` / `#C6C3CD`
- Lines: `#E7E4DD` (warm hairline)
- Accent: `hsl(225 65% 50%)` — blue, used sparingly for AI actions
- Warning: `#b45309` (amber)
- Critical: `#b91c1c` (red)
- Success: `#15803d` (green)
- Suggestion: `#1d4ed8` (blue)

## Typography (established)
- UI: Geist (sans-serif), 15px base, -0.01em tracking
- Document body: Georgia (serif), 15px, 1.7 line-height
- Display: Aeonik Medium / Urbanist, tight tracking

## Components (established)
- Buttons: pill-shaped (border-radius: 999px), dark ink primary, outline secondary
- Cards: white background, subtle shadow, 8px radius
- Chips: pill-shaped, warm chip background (#EDEAE3)
- Avatar: gradient circle
- Nav: pill-style links with active state

## Tunable Parameters
```yaml
DESIGN_VARIANCE: 3     # Predictable, centered layouts — clinical tool needs trust
MOTION_INTENSITY: 4    # Subtle transitions, no flashy animations
VISUAL_DENSITY: 5      # Balanced — not too sparse, not too dense
TONE: minimalist       # Warm monochrome, stripped to essence
```

## Framework
- Next.js 16 (App Router) + React 19 + Tailwind CSS 4
- shadcn/ui (customized to match warm aesthetic)
- Plate.js (for document editor workspace)

## Key Pages
1. `/generate` — Notes input (DONE — warm centered layout)
2. `/workspace/[id]` — Plate-powered document editor (NEXT — needs frontend-max treatment)
3. `/reports` — Report history list
4. `/settings` — Exemplar upload

## What Makes This Unforgettable
The document workspace should feel like editing a real clinical document — paper-like, serif typography, professional enough that the DOCX export looks identical to what's on screen. The AI refinement toolbar (select text → "Refine with AI") should feel like magic — a sparkle icon, smooth streaming, green highlight for suggestions. The NDIS planner flags should feel like expert peer review annotations, not error messages.
